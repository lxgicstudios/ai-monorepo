# Monorepo management and build optimization tools

Monorepo generator. Sets up workspace structure with shared configuration and tooling.

[![npm version](https://img.shields.io/npm/v/ai-monorepo.svg)](https://www.npmjs.com/package/ai-monorepo)
[![npm downloads](https://img.shields.io/npm/dm/ai-monorepo.svg)](https://www.npmjs.com/package/ai-monorepo)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![GitHub stars](https://img.shields.io/github/stars/lxgicstudios/ai-monorepo)](https://github.com/lxgicstudios/ai-monorepo/stargazers)
[![TypeScript](https://img.shields.io/badge/TypeScript-5.0+-blue)](https://www.typescriptlang.org/)



Set up a Turborepo monorepo structure with AI.

## Install

```bash
npm install -g ai-monorepo
```

## Usage

```bash
npx ai-monorepo
# Sets up Turborepo in current directory

npx ai-monorepo ./my-project
# Custom target directory
```

## Setup

```bash
export OPENAI_API_KEY=sk-...
```
